# todoGuide

A GitBook documenting the vibestack framework — an opinionated convention system for AI-assisted development with Claude Code.

The book walks developers through installing vibestack, understanding its core files (CLAUDE.md, TODO.md, ops.sh, docs/), using skills and slash commands, and scaling with squad mode.
